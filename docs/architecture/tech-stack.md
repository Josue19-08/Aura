# Tech Stack

## Blockchain

### Avalanche
- **C-Chain (EVM)** for smart contracts
- **Fuji Testnet** for development
- **Mainnet** for production

### Solidity ^0.8.24
- Smart contract language
- OpenZeppelin for security standards

### Hardhat
- Development environment
- Testing framework
- Deployment automation

## Backend

### Node.js + Express
- REST API server
- Lightweight and fast

### ethers.js v6
- Blockchain interaction
- Contract calls and events

### IPFS
- Decentralized metadata storage
- Options: web3.storage or Pinata

## Frontend

### React 18 + Vite
- Modern UI framework
- Fast development builds

### TailwindCSS
- Utility-first styling
- Custom brand theme

### RainbowKit + wagmi
- Wallet connection
- Web3 hooks

### Additional
- **Framer Motion** — Animations
- **React Router** — Navigation
- **QR Scanner** — Code scanning

## Deployment

- **Smart Contracts** → Avalanche (Fuji/Mainnet)
- **Backend** → Railway
- **Frontend** → Vercel
- **IPFS** → Pinata/web3.storage

## Development Tools

- **TypeScript** (optional)
- **ESLint + Prettier** — Code quality
- **GitHub** — Version control
- **Hardhat** — Testing and deployment
