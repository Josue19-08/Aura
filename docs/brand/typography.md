# Typography

## Font Families

### Space Grotesk
**Role:** Display / Hero text
**Weights:** 400, 500, 700
**Usage:** Large titles, product name, hero sections

### IBM Plex Mono
**Role:** Technical data / Code
**Weights:** 400, 500, 600
**Usage:** Hashes, lot IDs, timestamps, on-chain data

### DM Sans
**Role:** Interface / Body text
**Weights:** 400, 500, 700
**Usage:** UI elements, buttons, forms, body text

## Type Scale

| Level | Size | Line Height | Weight | Family | Usage |
|-------|------|-------------|--------|---------|-------|
| H1 | 48px | 1.1 | 700 | Space Grotesk | Page titles |
| H2 | 32px | 1.2 | 700 | Space Grotesk | Section titles |
| H3 | 24px | 1.3 | 600 | DM Sans | Subsection titles |
| Body | 16px | 1.6 | 400 | DM Sans | Standard text |
| Code | 14px | 1.6 | 500 | IBM Plex Mono | Technical data |

## Font Loading

```html
<link href="https://fonts.googleapis.com/css2?family=DM+Sans:wght@400;500;700&family=IBM+Plex+Mono:wght@400;500;600&family=Space+Grotesk:wght@400;500;700&display=swap" rel="stylesheet">
```
