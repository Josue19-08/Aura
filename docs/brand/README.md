# Brand Guidelines

## Overview

Aura's brand identity is built on three core principles:

1. **Trust** — Unbreakable verification
2. **Precision** — Technical accuracy
3. **Radical Transparency** — Nothing to hide

This is not a mass consumer app nor a cold corporate product — it's infrastructure that feels human and accessible.

## Quick Reference

### Primary Colors
- **Void** `#0A0A0F` - Main background
- **Signal** `#00E5CC` - Primary accent, verification, success
- **Caution** `#FF6B35` - Alerts, counterfeit warnings

### Typography
- **Display:** Space Grotesk
- **Technical Data:** IBM Plex Mono
- **Interface:** DM Sans

### Logo Concept
The letter "A" with an integrated tracking pin at its apex, evoking location tracking and product traceability.

---

## Design Philosophy

**Dark foundation, bright signals.** The dark background (`Void`) provides technical neutrality. Teal-green (`Signal`) as verification color evokes terminals, data, digital trust. Orange (`Caution`) for counterfeiting alerts — urgent without being blood red.

**Monospace for immutability.** Using `IBM Plex Mono` for on-chain data reinforces the idea of immutability and technical precision.

**Clean, accessible interface.** `DM Sans` keeps the UI approachable and clean, balancing the technical weight.
