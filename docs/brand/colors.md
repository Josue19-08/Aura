# Color Palette

## Core Colors

### Void
**Hex:** `#0A0A0F`
**RGB:** `rgb(10, 10, 15)`
**Usage:** Main background

### Signal
**Hex:** `#00E5CC`
**RGB:** `rgb(0, 229, 204)`
**Usage:** Primary accent, verification success

### Caution
**Hex:** `#FF6B35`
**RGB:** `rgb(255, 107, 53)`
**Usage:** Alerts, counterfeit warnings

## Supporting Colors

### Slate
**Hex:** `#1C1C2E`
**Usage:** Secondary backgrounds, cards

### Fog
**Hex:** `#8888AA`
**Usage:** Secondary text, labels

### White
**Hex:** `#F0F0F5`
**Usage:** Primary text

## CSS Variables

```css
:root {
  --color-void: #0A0A0F;
  --color-signal: #00E5CC;
  --color-caution: #FF6B35;
  --color-slate: #1C1C2E;
  --color-fog: #8888AA;
  --color-white: #F0F0F5;
}
```
