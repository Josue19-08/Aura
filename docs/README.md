## Project Overview

Aura is a blockchain-based product traceability system built on Avalanche. Each product leaving a factory receives a unique, immutable digital identity registered on the blockchain. Throughout the entire chain—manufacturer, distributor, importer, point of sale—every movement is recorded on-chain.

Any actor in that chain, from distributor to end buyer, can verify at any time whether the product is authentic, which batch it came from, when it was manufactured, and where it passed through before reaching their hands.

---

## Key Features

- ✅ **Immutable Product Registry** - Permanent on-chain record
- 🔄 **Custody Chain Tracking** - Complete transfer history
- 🔍 **Public Verification** - Anyone can verify authenticity
- 🚨 **Anti-Counterfeiting** - Automatic fraud detection
- 📊 **Full Transparency** - Complete product lifecycle visibility
